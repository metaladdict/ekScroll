This is a very simple scroll plugin
  • Create a div
  • Add your content
  • Configure it with CSS (LESS file included)
  • Execute $('.demoDiv').ekScrollable();
  • That's it

This plugin works with touch devices

It depends on :

  • jquery
  • jquery UI / draggable
  • jquery.mousewheel.js of Brandon Aaron
